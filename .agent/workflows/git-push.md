## Git Push Workflow

// turbo-all

1. Stage all changes:
```bash
cd /Users/taps/Desktop/MDRPEDIA/mdrpedia && git add .
```

2. Commit with a descriptive message:
```bash
cd /Users/taps/Desktop/MDRPEDIA/mdrpedia && git commit -m "<message>"
```

3. Push to GitHub:
```bash
cd /Users/taps/Desktop/MDRPEDIA/mdrpedia && git push
```

### Notes
- Remote URL should be configured with a PAT for authentication
- Branch: `main`
